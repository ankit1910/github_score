This exercise is created to calculate the github score of a particular user.

Assumptions:
  1. Github handler provided to the system is correct and exists.
  2. Score of all other events(mentioned in commit.rb) is considered as 1.
  3. System has a working internet connection,

Usage:
  For calcualte score run the exercise.rb inside bin directory to checkout score for particular github handler(dhh by default).

  You can change the handler variable inside 'exercise.rb' to check other's score.

Example:
  `ruby <path to bin folder>/exercise.rb`
